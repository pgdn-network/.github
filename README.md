# PGDN .github Community Health Files

This repository defines **organization-wide community standards** for the [`pgdn-network`](https://github.com/pgdn-network) GitHub organization.  
Any file here automatically applies to all repositories in the org.

## What’s Included
- **CONTRIBUTING.md** — how to contribute across repos (code, research, docs).  
- **CODE_OF_CONDUCT.md** — expected behavior in the community.  
- **SECURITY.md** — responsible disclosure policy (Discord-only reporting).  
- **SUPPORT.md** — how to get help (Discord, issues, etc).  
- **GOVERNANCE.md** — lightweight governance model.  
- **ISSUE_TEMPLATE/** — structured templates for bugs, features, and security.  
- **PULL_REQUEST_TEMPLATE.md** — PR checklist and guidance.  
- **FUNDING.yml** — optional funding links.

## Contact
- **Discord:** [join the community](https://discord.com/invite/fJx4fCPN), ask questions, or report issues privately.  
- **Research & Reports:** see [pgdn.ai](https://pgdn.ai) and the [research hub](https://pgdn.ai/research).  
- **GitHub Issues:** for non-security bugs and feature requests, use the appropriate repo.

## Security
Do **not** open public issues for vulnerabilities. Report them privately via Discord.  
See [SECURITY.md](SECURITY.md) for details.
